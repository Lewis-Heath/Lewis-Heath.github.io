Pacman

Pacman is a C++ project using S2D. It contains many of the feautres of the orignal Pacman game such as the ghosts, Pacman, Munchies, Cherries but also has a few of my own twists added to it. For example a Speed boost. The objective of the game is still the same - Collect all the munchies and dont die to the ghosts!

Pairs

Pairs is a simple input game written in C++. The program shows the user cards labelled 1-10 which, all have randomly assigned letters the user cannot see. The user then inputs 2 cards they want to flip, if the cards are the same letter they stay flipped, if the letters on the cards are different the cards flip back over and display numbers again. The objective of the game is to flip all 10 cards in the shortest turns possible.

Rock Paper Scissors

Just like how its played in real life, this C++ project takes a users input - Rock, Paper or Scissors and matches it against a randomly generated Rock, Paper or Scissors from the program. A win, loss or draw is then announced. The aim of the game is to beat the computer 3 times before it beats you 3 times.
